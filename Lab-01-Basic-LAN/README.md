# Lab 01 — Basic LAN

## Objective

Build a basic local area network by connecting two PCs through a Cisco 2960 switch.

## Devices

- 1 Cisco 2960 switch
- 2 PCs
- 2 copper straight-through cables

## IP Addressing

| Device | IP Address | Subnet Mask |
|--------|------------|-------------|
| PC0 | 192.168.10.10 | 255.255.255.0 |
| PC1 | 192.168.10.20 | 255.255.255.0 |

## Testing

Both computers successfully communicated with 0% packet loss.

## What I Learned

- A switch connects devices inside the same LAN.
- A switch forwards frames using MAC addresses.
- ARP maps an IPv4 address to a MAC address.
- Devices on different networks need a router to communicate.
